# Installation #

1. Clone the repo to ~/configs.
2. Create symlinks and install deps based on the following sections.

## urxvt terminal ##

deps: urxvt-tabbedex

    ln -s ~/configs/Xdefaults ~/.Xdefaults

## awesome ##

    ln -s ~/configs/awesome ~/.config/awesome

## git ##

    ln -s ~/configs/gitconfig ~/.gitconfig
